# ADC Polling (STM32)

## Platform
STM32 B-L4S5I-IOT01A

## Description
Implemented ADC in polling mode to read analog sensor data using STM32 microcontroller.

## Peripherals Used
- ADC
- GPIO (if used)

## Working
The ADC continuously reads analog input and processes the data in polling mode.

## Output
Sensor values observed via serial output / debugger
